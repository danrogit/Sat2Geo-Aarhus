# Data and Imagery Notice

The code in this repository is MIT licensed.

Satellite imagery, map tiles, generated chips, embeddings, and vector indexes may be subject to separate provider licenses. Do not publish or redistribute imagery-derived files unless you have the right to do so.

This is especially important for commercial map providers. Google Maps Platform documentation restricts bulk tile access, caching, indexing, and storage outside permitted use cases. If you use Google, Mapbox, Esri, national mapping agencies, or any other provider, read and follow that provider's terms.

For an openly redistributable dataset, prefer open imagery or imagery you own, and include the correct attribution and license text.

Files that may be imagery-derived:

```text
data/chips/
data/index/
data/embeddings/
examples/
```

If you are unsure about the imagery license, publish the code and scripts only, then ask users to build their own local database.
